# 🚀 دليل سريع لرفع المشروع على GitHub

## الخطوة 1: تثبيت Git
1. **اذهب إلى:** https://git-scm.com/download/win
2. **حمل Git for Windows**
3. **ثبّت البرنامج وأعد تشغيل الكمبيوتر**

## الخطوة 2: إنشاء حساب GitHub
1. **اذهب إلى:** https://github.com
2. **سجل حساب جديد أو سجل دخول**
3. **أنشئ مستودع جديد:**
   - Name: `ecommerce-store`
   - Description: `متجر إلكتروني احترافي`
   - Public/Private: اختر حسب احتياجك

## الخطوة 3: تشغيل السكريبت
1. **افتح Command Prompt كـ Administrator**
2. **اذهب إلى مجلد المشروع:**
   ```cmd
   cd "C:\Users\ALBAHA\Desktop\my store"
   ```
3. **شغل السكريبت:**
   ```cmd
   UPLOAD_TO_GITHUB.bat
   ```
4. **أدخل اسم مستخدم GitHub عندما يطلب**

## الخطوة 4: التحقق
1. **اذهب إلى رابط المستودع**
2. **تأكد من وجود جميع الملفات**
3. **تحقق من README.md يظهر بشكل صحيح**
