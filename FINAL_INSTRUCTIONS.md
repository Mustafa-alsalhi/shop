## 🚀 خطوات الرفع النهائية:

### **الطريقة 1: السكريبت الآلي (موصى بها)**
1. **ثبّت Git من:** https://git-scm.com/download/win
2. **افتح Command Prompt كـ Administrator**
3. **شغل:** `UPLOAD_TO_GITHUB.bat`
4. **أدخل اسم مستخدم GitHub**

### **الطريقة 2: يدوياً**
1. **ثبّت Git**
2. **افتح Command Prompt**
3. **اذهب للمجلد:** `cd "C:\Users\ALBAHA\Desktop\my store"`
4. **نفذ الأوامر:**
   ```cmd
   git init
   git add .
   git commit -m "Complete e-commerce store"
   git remote add origin https://github.com/YOUR_USERNAME/ecommerce-store.git
   git push -u origin main
   ```
